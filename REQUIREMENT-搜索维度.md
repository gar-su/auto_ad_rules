# 需求文档：短剧范围搜索维度

## 版本

| 版本 | 日期 | 作者 | 变更 |
|------|------|------|------|
| v1.0 | 2026-05-27 | - | 初稿 |

## 背景

短剧范围模块的搜索栏仅支持按短剧名称搜索。实际业务中，运营人员常需通过剧本编号定位短剧（剧本编号来自内部剧本管理系统，如 `SC20240501001`），现有单维度搜索无法满足需求。

## 目标

在短剧范围搜索栏增加搜索维度选择，支持按短剧名称、剧本编号、剧本名称三种维度切换搜索，并相应扩展表格列展示剧本编号。

## 范围

- **涉及页面**：控损规则创建 — 控损对象 — 短剧范围
- **涉及文件**：`index.html`
- **不涉及**：账户范围、推广链接的搜索栏

---

## 功能详述

### F1. 搜索维度选择器

| 属性 | 描述 |
|------|------|
| 位置 | 搜索栏最左侧，搜索输入框之前 |
| 形态 | `<select>` 下拉框 |
| 选项 | 短剧名称 / 剧本编号 / 剧本名称 |
| 默认值 | 短剧名称 |
| 样式 | 与搜索输入框视觉连接（共用 border，无间隙） |

状态示意：

```
[短剧名称 ▾][________________________][搜索]
[剧本编号 ▾][________________________][搜索]
[剧本名称 ▾][________________________][搜索]
```

### F2. 输入框占位提示联动

| 维度 | placeholder |
|------|-------------|
| 短剧名称 | 请输入短剧名称 |
| 剧本编号 | 请输入剧本编号 |
| 剧本名称 | 请输入剧本名称 |

切换维度时，输入框 placeholder 即时更新。

### F3. 剧本编号列

| 属性 | 描述 |
|------|------|
| 位置 | 左侧候选表格，checkbox 与剧本名称之间 |
| 列头 | 剧本编号 |
| 数据 | 剧本编号字符串，如 `SC20240501001` |
| 联动 | 右侧已选表格同步展示剧本编号列 |

左侧候选表格列变更：

```
变更前:  ☐ | 剧本名称 | 短剧名称 | 语种
变更后:  ☐ | 剧本编号 | 剧本名称 | 短剧名称 | 语种
```

右侧已选表格列变更：

```
变更前:  剧本名称 | 语种 | 操作
变更后:  剧本编号 | 剧本名称 | 语种 | 操作
```

### F4. 转写逻辑适配

左表选入右表时，左侧有 5 列（含 checkbox 和 4 个数据列），右侧仅展示 3 个数据列（剧本编号、剧本名称、语种），需跳过左侧"短剧名称"列。

数据映射：

```
左表 col(1) 剧本编号 → 右表 col(1) 剧本编号
左表 col(2) 剧本名称 → 右表 col(2) 剧本名称
左表 col(4) 语种     → 右表 col(3) 语种
左表 col(3) 短剧名称 → (跳过，不转入右表)
```

处理规则：当左侧行数据列数 > 右侧数据列数时，取前 (N-1) 列 + 末列，丢弃中间多余列。

## 交互清单

| ID | 触发 | 行为 |
|----|------|------|
| I-DIM-1 | 下拉切换维度 | 搜索输入框 placeholder 即时更新 |
| I-DIM-2 | checkbox 勾选行 | 新列结构下的数据转入右侧（含剧本编号，跳短剧名称） |

## 验收标准

- [ ] **AC1**：搜索栏左侧出现维度下拉框，含"短剧名称""剧本编号""剧本名称"三个选项
- [ ] **AC2**：下拉框与搜索输入框视觉上连为一体（无左/右 border-radius 间隙）
- [ ] **AC3**：切换维度后，输入框 placeholder 同步变化
- [ ] **AC4**：左侧候选表格新增"剧本编号"列，位于 checkbox 和"剧本名称"之间
- [ ] **AC5**：右侧已选表格新增"剧本编号"列，位于"操作"列之前
- [ ] **AC6**：勾选行后右侧正确显示剧本编号、剧本名称、语种（短剧名称不出现）
- [ ] **AC7**：原有转写逻辑（账户范围、推广链接）不受影响

## 技术实现要点

| 项目 | 说明 |
|------|------|
| 维度选择器 CSS | `.search-dim`：border-radius:4px 0 0 4px, border-right:none, min-width:100px |
| 输入框衔接 CSS | `.search-dim + .search-input`：border-radius:0, border-left:none |
| placeholder 切换 JS | 监听 `.search-dim` change 事件，根据 value 映射 placeholder 文案 |
| 转写列跳过 JS | `data.length > rightCols` 时取 `[...data.slice(0, rightCols-1), data[data.length-1]]` |
| 表格列新增 | thead 和 tbody 中每个 tr 追加一个 `<td>` |
